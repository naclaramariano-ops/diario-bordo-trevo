PASSO A PASSO — DIÁRIO DE BORDO TREVO NO CLOUDFLARE PAGES

1. SUPABASE
- Acesse supabase.com.
- Crie um projeto.
- Nome sugerido: diario-bordo-trevo.
- Aguarde o projeto ficar pronto.

2. CRIAR TABELAS
- Vá em SQL Editor.
- Abra o arquivo sql/schema_supabase.sql deste pacote.
- Copie todo o conteúdo.
- Cole no SQL Editor do Supabase.
- Clique em Run.

3. COPIAR CHAVES DO SUPABASE
- Vá em Project Settings > API.
- Copie a Project URL. Use sem /rest/v1/.
- Copie a chave publicável / anon public key.

4. CONFIGURAR O APP
- Abra assets/app.js.
- Substitua:
  SUPABASE_URL: "COLE_AQUI_SUA_SUPABASE_URL"
  SUPABASE_ANON_KEY: "COLE_AQUI_SUA_SUPABASE_ANON_KEY"
- Salve o arquivo.

5. SUBIR NO CLOUDFLARE PAGES
- Acesse dash.cloudflare.com.
- Entre em Workers & Pages.
- Clique em Create application.
- Escolha Pages.
- Se for upload manual, selecione Upload assets.
- Arraste a pasta inteira do app ou o ZIP descompactado.
- Build command: deixe em branco.
- Build output directory: / ou .
- Clique em Deploy.

6. ACESSAR NO CELULAR
- Abra o link gerado pelo Cloudflare Pages no navegador do celular.
- Primeiro acesso precisa ter internet.
- Android: Chrome > três pontinhos > Adicionar à tela inicial.
- iPhone: Safari > Compartilhar > Adicionar à Tela de Início.

7. TESTE OFFLINE
- Abra o app online uma vez.
- Desligue a internet.
- Abra o app novamente.
- Crie um diário.
- O app mostra pendente de sincronização.
- Ligue a internet.
- Clique em Sincronizar agora ou aguarde sincronizar automaticamente.
