# Diário de Bordo Trevo — Enterprise V10

## Escopo
- Correção de respostas neutras: não geram Atenção.
- Resumo operacional determinístico na aba Hoje.
- Cards operacionais permanecem fechados e expansíveis.
- Histórico analítico por Dia, Semana e Mês.
- Comparação com período anterior, ranking de causas e máquinas.
- Estrutura modular em `src/modules` e `src/shared`.
- Sem API externa de IA: resumo inteligente local, gratuito e auditável.

## Build
Executar `npm ci` e `npm run build`.
