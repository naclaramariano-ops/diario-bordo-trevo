# V6.8.2 — Múltiplos rascunhos por Data + Turno + Área

- A passagem passa a ser identificada por Data + Turno + Área.
- Envase 1 e Envase 2 podem ser preenchidos simultaneamente por usuários diferentes.
- Turnos e datas diferentes não se bloqueiam.
- Datas passadas continuam permitidas; datas futuras permanecem bloqueadas.
- Um usuário pode possuir vários rascunhos em andamento.
- A aba Novo lista as passagens em andamento do próprio usuário.
- Somente a combinação idêntica Data + Turno + Área é bloqueada quando pertence a outro responsável.
- O conteúdo do rascunho é recuperável em outro dispositivo pelo mesmo usuário.
- A finalização atualiza o mesmo registro, sem criar duplicidade.
