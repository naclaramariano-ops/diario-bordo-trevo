# Diário de Bordo Trevo — V6.3

## Alterações

- Mantida a base V5 Enterprise Foundation e o módulo V6.2 de Envase.
- Preservada a regra atual de classificação automática: Conforme, Atenção e Desvio.
- Aba +Novo ajustada para melhor responsividade e preenchimento guiado.
- Campos de SKU convertidos para lista suspensa com opções do Excel enviado pelo usuário.
- Campos de seleção avançam automaticamente para o próximo campo após escolha.
- Validação mantém todos os campos obrigatórios e direciona para a máquina pendente.
