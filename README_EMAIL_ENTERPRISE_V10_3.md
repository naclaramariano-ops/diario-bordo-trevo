# Enterprise V10.3 — Template visual do e-mail

Esta atualização não altera nenhuma regra do aplicativo, banco, Edge Functions, autenticação ou recuperação de senha.

## Arquivo atualizado

`supabase/apps-script/Code.gs`

## Publicação no Google Apps Script

1. Abra o projeto **Diário de Bordo Trevo - Envio de E-mail**.
2. Abra o arquivo principal `Code.gs`.
3. Substitua todo o conteúdo pelo arquivo incluído neste pacote.
4. Clique em **Salvar**.
5. Clique em **Implantar > Gerenciar implantações**.
6. Edite a implantação existente.
7. Em **Versão**, selecione **Nova versão**.
8. Clique em **Implantar**.

A URL `/exec` permanece a mesma quando a implantação existente é atualizada.
